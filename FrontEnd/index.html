How ratings work (chess-style)

  • Each player starts around 1200 rating.
  • Win vs higher-rated player ‚Üí bigger rating gain.
  • Win vs lower-rated player ‚Üí smaller gain.
  • Lose vs lower-rated player ‚Üí bigger rating loss.
  • Draw / close scores ‚Üí small adjustments both sides.

Behind the scenes, an Elo-like formula updates rating based on opponent strength, quiz difficulty, and final score‚Äîsimilar to chess.com rating behaviour. :contentReference[oaicite:0]{index=0}

Mode-based ratings

  • Quiz (Rapid) ‚Äì standard class quizzes.
  • Tournament (Blitz) ‚Äì fast-paced event games.
  • Practice (Casual) ‚Äì practice mode, rating optional.

Just like chess.com has Rapid / Blitz / Bullet, Samarpan can track separate ratings per mode, plus a global rating for ranking.
